# Database Systems Management Project -- Injury Tracker

## Overview
This repository contains the project developed for the **Database** course.  
The project focuses on designing and implementing a **web-based information system from scratch**, including both frontend and backend components.

The system was built covering the full workflow from database design to deployment.

---

## Objectives
- Design a complete database-driven web application
- Apply ER modeling and relational schema design
- Implement frontend and backend integration
- Use SQL for data storage and querying
- Consider security and maintenance performance aspects

---

## Learning Outcomes
- Read and design **Entity-Relationship (ER) diagrams**  
- Create and normalize **relational database schemas**  
- Write and evaluate **SQL queries**  
- Understand **query processing and optimization**  
- Apply **transactions and database consistency concepts**  
- Identify **security and privacy issues in databases**  
- Develop **database-backed web applications**

---

## 🛠️ Technologies Used
- **MySQL** – Database management  
- **PHP / Javascript** – Backend development  
- **HTML / CSS** – Frontend interface
- **Linux** – Development environment
